# Todo List App

A simple, elegant todo list application with drag-and-drop functionality and persistent storage. Built with vanilla JavaScript, HTML, and CSS.

## Features

- **Add, Edit, Delete Tasks** - Full CRUD operations for managing your todos
- **Drag and Drop** - Reorder tasks by dragging them to a new position
- **Task Counter** - View total tasks and completion count at a glance
- **Mark as Complete** - Check off completed tasks with visual strikethrough
- **Local Storage** - All tasks are saved automatically and persist between sessions
- **Responsive Design** - Clean, modern UI that works on all devices
- **No Dependencies** - Pure vanilla JavaScript, no frameworks required

## Demo

Try it live: [https://achab1.github.io/todo-app/](https://achab1.github.io/todo-app/)

## Usage

1. **Add a task**: Type in the input field and click "Add" or press Enter
2. **Complete a task**: Click the checkbox next to any task
3. **Edit a task**: Click on the task text to edit it inline
4. **Delete a task**: Click the "Delete" button
5. **Reorder tasks**: Click and drag any task to move it up or down

## Installation

No installation required! Simply open `index.html` in your web browser.

```bash
# Clone the repository
git clone https://github.com/achab1/todo-app.git

# Navigate to the directory
cd todo-app

# Open in browser
start index.html  # Windows
open index.html   # macOS
xdg-open index.html  # Linux
```
